# Codex Spec: 09-cell-view

## Purpose
Create the cell type detail view — the deepest drill-down level showing full marker information for a single cell type, including positive/negative markers for both species, aliases, references, and external links.

## Dependencies
- `05-datastore.md` (cell type data)
- `06-router.md` (navigation)
- `11-compare.md` (add to compare)
- `12-export.md` (export single cell type)
- `14-links.md` (external database links)

## Output Files
- `js/cell-view.js`

## API

```javascript
class CellView {
  constructor(container: HTMLElement, datastore: CellMarkersDatastore, router: Router)

  render(route: Route): void
  activate(): void
  deactivate(): void
}
```

## Marker Table Design

### Positive Markers
- Display as a grid of tags/chips
- Each tag is clickable → searches for other cell types with that marker
- Color: `var(--marker-positive)` background (#E9ED4C)
- Hover: show tooltip with full marker name (if abbreviated)

### Negative Markers
- Display as a grid of tags/chips
- Color: `var(--marker-negative)` background (#FF9400)
- Less prominent than positive (slightly smaller, muted)

### Species Toggle
- Show both species side-by-side on desktop
- On mobile: tabbed view (Human / Mouse tabs)
- Active species tab matches global species toggle

## Marker Tag HTML
```html
<button class="marker-tag positive" data-marker="CD34" data-species="human"
        title="Click to find other cell types expressing CD34">
  CD34
</button>
```

Clicking a marker tag:
1. Navigates to search view with that marker as query
2. Shows all cell types expressing that marker

## External Links Section
For each positive marker, generate links to:
- **UniProt**: `https://www.uniprot.org/uniprot/?query=gene:{SYMBOL}+organism:{SPECIES_ID}`
  - Human organism ID: 9606
  - Mouse organism ID: 10090
- **CellMarker 2.0**: `http://bio-bigdata.hrbmu.edu.cn/CellMarker/search/{SYMBOL}`
- **PanglaoDB**: `https://panglaodb.se/search.html?query={SYMBOL}`

Links are generated by the `links.js` module (spec 14).

## Compare Button
```html
<button class="compare-add-btn" data-cell-type-id="hepatocyte">
  <span class="compare-icon">+</span> Add to Compare
</button>
```
- When cell type is already in compare tray: button shows "In Compare" (disabled style)
- Click adds cell type to compare tray (spec 11)

## Export Button
- Exports this cell type's markers as CSV (spec 12)

## References
- Display reference numbers from `cellType.references` array
- Map to source entries from `metadata.sources`
- Show as numbered list with source title and DOI link

## Provenance
- Show `source`, `added_in_version`, `last_modified_version`
- Small, muted text at bottom

## Edge Cases
- Cell type not found: show error with link back to organ
- No markers for selected species: show "No markers available for [species]" message
- Cell type with many markers (>20): show all, with scroll if needed
- Marker tag click on mobile: navigate to search (no hover)
